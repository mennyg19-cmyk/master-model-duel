# P7 Quality Review — arm-03 (blind)

**Phase:** P7 — Package engine live
**Reviewer:** external quality specialist
**Tree:** `arms/arm-03/workspace`
**EXPECTED:** `shared/phases/PHASE-P7-EXPECTED.md`
**Smoke claim:** 16/16 PASS (`arms/arm-03/results/PHASE-P7-SMOKE.md`)

Scope: correctness, broken flows, stubs, missing smoke, regressions vs EXPECTED. Findings only — no fixes.

## Summary

- **Findings:** 14 (3 high, 6 medium, 5 low)
- **High:** PICKUP/SHIP terminal-stage invariant not enforced on live paths; greeting-card / label PDFs produced at LETTER size instead of card stock / 4×6; two competing PDF renderers with the live one producing invalid PDFs for non-ASCII content.
- **Medium:** large dead-code surface (duplicate `packages/actions.ts`, 4 dead routes, dead `FulfillmentActions`, dead `print/batches.ts` batch fns, dead `packageStagesForBatch`); UI reads `json.packageStages` which the API never returns; channel bulk-stage skips global `AuditLog`; `reprint-group` includes already-shipped packages.
- **Low:** status note claims ASCII `->` but dead code still has Unicode `→`; orphaned `packing-slip` and `print-artifacts` routes with no UI link; empty greeting-card artifacts for greetingless groups; smoke gaps (regroup, bulk dashboard actions, per-group separation, card-stock/label sizes, method/stage invariant).
