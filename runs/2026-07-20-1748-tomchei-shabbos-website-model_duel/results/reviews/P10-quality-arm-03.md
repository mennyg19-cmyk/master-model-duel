## Summary

- EXPECTED coverage: items 1–4 are nominally implemented and smoke passes 3/3, but four high-severity defects (Q-01..Q-04) produce wrong draft data or broken staff UI on the very flows the smoke claims to cover — smoke assertions are too narrow to catch them.
- Replacement chain (R-048/G-013): BFS + price-smart default works; dead `pickPriceSmart` helper (Q-14).
- Customer repeat review (UR-007/G-011/G-012): two-checkbox confirm + backend guards work; add-ons and option adjustments are silently dropped (Q-01, Q-02).
- Staff single/bulk repeat (R-057/R-058): bulk path works; single path mishandles `needsReview` (Q-04); source-version bump in bulk is inconsistent with single (Q-07).
- New-season wizard + gate + auto-flip (R-097/UR-008): wizard creates CLOSED season and gate switch works, but copy-from-season is UI-unreachable (Q-08), inventory/BOM not copied (Q-11), close schedule unreachable (Q-10), schedule input doesn't show existing value (Q-09), and a duplicate dead cron route exists (Q-12).
- Archive browsable off-season: works (`archive/[slug]` + `archive/[slug]/[productSlug]` enforce CLOSED + `storeOpen=false`).
