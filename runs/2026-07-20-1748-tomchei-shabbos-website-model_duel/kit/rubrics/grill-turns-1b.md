**Definitions**

- **Needed vs fluff:** 2 = clearly reduces ambiguity; 0 = already answered / filler  
- **Explain-down:** 2 = non-jargon human can answer; 0 = unexplained jargon wall  
- **Options real:** 2 = options fit this product; 0 = hallucinated / inapplicable  
- **Uptake:** 2 = next move shows hearing the user (incl. “use recommended”); 0 = ignored  
- **Faithful capture:** 2 = “I heard” matches user; 0 = silent rewrite  

**Fluff turns:** list turn #s (excluded from mean “needed”, or score 0).

**turn_quality_mean** = mean of (needed, explain-down, options, uptake, faithful) over non-fluff turns (or all turns with fluff needed=0).

**necessary_turns** = count non-fluff.

**Recommended-used rate** (fact only): __% user picked model’s Recommended.
